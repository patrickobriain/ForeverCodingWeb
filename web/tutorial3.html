How to add an image to your HTML website

Without images, the internet would be a pretty dull place.
 
So why should your website be any different?
 
In this tutorial, we’ll show you how to add images to your website.
 
Let’s use the HTML file we made on the last tutorial.
 
Open it in Notepad (see below)

So, we are going to add our image after the 2 paragraphs.
 
You will need to insert this <img src="ourlogo.jpg" alt=Logo"> tag.
 
You will need to edit the ourlogo part, depending on what your image is called. You may need to edit the file type .jpg, again depending on what type of image it is. 
The image must be stored in the same location as the HTML file (as above) in order for this to work.
 
We are going to use the ourlogo.jpg file that is stored in that folder.
 
Enter this <img src="ourlogo.jpg" alt=Logo"> code depending on where you want it, but it has to be after the <body> tag.

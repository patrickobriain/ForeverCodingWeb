How to add a radio button to your HTML page

It's really simple to add a radio button to your HTML page.

We're going to say the question is Example Question and the options are Option 1 and Option 2, but of course you can adjust it to your own questions. 

Just add this line of code to your HTML page. 

<label> Example Question </label>

<input type="radio" name="option">Option 1

<input type="radio" name="option">Option 2

<br><br>

Now, save your file and make sure to include the html extension.

 

You can see how to save the file on our first HTML tutorial here.

 

Now you know how to add a radio button to your HTML page.
